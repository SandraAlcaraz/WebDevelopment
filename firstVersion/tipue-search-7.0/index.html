Tipue Search, a jQuery Site Search Plugin

Version 7.0
Tipue Search is a site search jQuery plugin. It's free, open source, responsive and fast. Tipue Search only needs a browser that supports jQuery. It doesn't need MySQL or similar. It doesn't even need a web server.
Tipue Search is released under the MIT License. It's free for both commercial and non-commercial use.

Tipue Search has full and expansive Help. We also offer a range of flexible support plans, including free.
